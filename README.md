# frontend-mentor-solutions
My solutions to [Frontend Mentor challenges] (https://www.frontendmentor.io/challenges), focusing on clean UI, best practices, and matching the provided designs as closely as possible. 


## Structure
Each folder in this repo represents the solution to a particular frontend mentor challenge and includes;
- The complete source code
- Assets provided by Frontend Mentor
- A finished implementation of the challenge

## Tech Stack
- HTML5
- CSS
- Sass
- Javascript
- Frameworks or libraries where appropriate

## Purpose

This repository serves as:
- A personal learning log
- A reference for frontend patterns and techniques
- A portfolio of real-world UI implementations

Suggestions and improvements are always welcome.
